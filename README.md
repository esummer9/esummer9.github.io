# Notion 데스크톱 요약

간단 소개
Notion 데스크톱은 데스크톱 환경에서 노트, 문서, 데이터베이스, 작업 관리를 통합해 사용하는 앱입니다. 브라우저 기반 서비스와 동기화되며 오프라인 편집을 지원합니다.

주요 기능
- 페이지와 블록 기반 편집: 텍스트, 제목, 표, 코드 블록, 이미지 등 다양한 블록을 끌어다 놓아 구성 가능.
- 데이터베이스: 테이블, 보드(Kanban), 캘린더, 갤러리 뷰로 정보 구조화 및 필터/정렬 가능.
- 템플릿: 반복적인 작업용 템플릿을 생성·적용 가능.
- 협업: 댓글, 멘션, 권한 설정으로 팀 협업 지원.
- 동기화 및 오프라인: 변경 내용은 로컬에 저장되고 온라인으로 다시 연결되면 동기화됨.

설치 및 시작
- 공식 Notion 웹사이트에서 macOS/Windows용 설치 프로그램을 내려받아 설치.
- 계정으로 로그인하면 기존 작업공간이 자동으로 동기화됨.

사용 팁
- 빠른 블록 생성: 슬래시(/)로 블록 메뉴 열기.
- 검색: 전체 페이지 검색(단축키로 빠르게 호출).
- 드래그 앤 드롭으로 페이지 구조 재배치.
- 자주 쓰는 템플릿과 데이터베이스 뷰를 저장해 재사용.

보안 및 설정
- 계정 보안(이중 인증 등)과 작업공간 권한을 설정하여 접근 제어.
- 앱 환경설정에서 동기화, 알림, 단축키 등을 조정 가능.

추가 도움말
- 상세 기능이나 문제 해결은 Notion 공식 도움말 문서와 지원 페이지를 참고하세요.
- 간단한 워크플로우 예시나 템플릿이 필요하면 알려주세요.
